# Driving Log Analysis 포트폴리오 (Python)

본 저장소는 차량 주행 로그 데이터를 기반으로  
급가속·급제동 이벤트를 탐지하고,  
이를 통해 운전자 주행 성향을 분석하는 데이터 분석 프로젝트 모음입니다.

자동차 시험·QA·주행 데이터 분석 관점에서  
실제 현업에서 사용할 수 있는 로직 구현을 목표로 합니다.
