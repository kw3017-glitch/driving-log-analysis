A# Driving Log Analysis 포트폴리오 (Python)

본 저장소는 차량 주행 로그 데이터를 기반으로  
급가속·급제동 이벤트를 탐지하고,  
이를 통해 **운전자 주행 성향 및 위험도를 분석**하는 데이터 분석 프로젝트 모음입니다.

단순 시각화가 아닌,  
**자동차 시험·QA·주행 데이터 분석 관점에서  
실제 현업에서 활용 가능한 로직 구현**을 목표로 설계되었습니다.

---

## 📂 프로젝트 구성 개요
주행 로그 데이터 분석 파이프라인을  
기초 분석 → 이벤트 탐지 → 성향 분류 → 점수화 모델로 단계적으로 확장합니다.

---

## 📁 프로젝트 목록

### 프로젝트 01. 주행 로그 기본 분석
- 주행 로그 데이터 구조 이해
- 시간 기반 속도 / 가속도 계산
- 기본 주행 패턴 시각화

📄 문서: `project-01-driving-log-analysis.md`

---

### 프로젝트 02. 급가속 · 급제동 이벤트 탐지
- 가속도 임계값 기반 이벤트 정의
- 급가속 / 급제동 이벤트 자동 탐지
- 이벤트 구간 시각화

📄 문서: `project-02-hard-event-detection.md`

---

### 프로젝트 03. 운전자 주행 성향 분류
- 급가감속 이벤트 빈도 기반 분석
- 이벤트 개수(Count) / 시간·거리 기반 빈도(Rate) 이중 분류
- 데이터 부족 시 과대평가 방지 로직 포함

📄 문서: `project-03-driver-behavior-classification.md`

---

### 프로젝트 04. 공격적 주행 점수 모델
- 다중 주행 지표 종합 점수화
- 급가감속 빈도, 급제동 비율, 속도 변동성 반영
- 운전자 공격성 점수(0~100) 산출

📄 문서: `project-04-aggressiveness-score.md`

---

### 프로젝트 05. ADAS 개입 필요도 점수 모델
- 기존 프로젝트 결과 통합 활용
- ADAS 개입 필요도를 하나의 점수로 정량화
- 개입 레벨(경고/제어) 판단 로직 설계

📄 문서: `project-05-adas-intervention-score.md`

---
