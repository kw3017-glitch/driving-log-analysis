# 프로젝트 03: 운전자 주행 성향 분류

## 목적
급가속·급제동 이벤트 빈도를 기반으로
운전자 주행 성향을 정량적으로 분류합니다.

## 분석 지표
- 급가속 이벤트 수
- 급제동 이벤트 수
- 총 급가감속 이벤트 수
- 분당 이벤트 수 (events/min)
- km당 이벤트 수 (events/km)

## 분류 방식

### 1. 이벤트 개수 기반 분류
- Smooth Driver
- Normal Driver
- Aggressive Driver

### 2. 이벤트 빈도 기반 분류
- 주행 시간 고려
- 데이터 부족 시 분류 제외 (Insufficient Data)

## 안정성 처리
- 짧은 주행 데이터 과대평가 방지
- 시간/거리 부족 시 빈도 지표 비활성화

## 출력 결과
- 이벤트 통계
- 빈도 지표
- 운전자 성향 분류 결과
