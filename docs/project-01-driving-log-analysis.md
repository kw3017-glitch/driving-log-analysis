# 프로젝트 01: 주행 로그 기본 분석

## 목적
차량 주행 로그 데이터를 분석하기 위한 기본 데이터 구조를 이해하고,
이후 이벤트 분석을 위한 기반을 구축합니다.

## 주요 내용
- 시간, 속도, 가속도 데이터 구성
- 속도 변화에 따른 가속도 계산
- 시간 대비 속도 변화 시각화
- CSV 저장 및 재로딩 검증

## 사용 데이터
- time_sec (초)
- speed_kmh (km/h)
- accel_mps2 (m/s²)

## 산출물
- 속도-시간 그래프
- 분석용 주행 로그 CSV

## 관련 파일
- driving_log_analysis.ipynb
- data/driving_log.csv
