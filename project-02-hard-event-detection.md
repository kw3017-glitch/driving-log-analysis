# 프로젝트 02: 급가속 · 급제동 이벤트 탐지

## 목적
가속도 임계값을 기준으로 비정상 주행 이벤트를 탐지합니다.

## 이벤트 정의
- 급제동 (Hard Braking)  
  가속도 < -3.0 m/s²
- 급가속 (Hard Acceleration)  
  가속도 > +3.0 m/s²

※ 임계값은 예시값이며 시험 기준에 따라 조정 가능

## 주요 기능
- 급가감속 이벤트 필터링
- 이벤트 발생 시점 추출
- 주행 그래프 상 이벤트 시각화
- 이벤트 발생 횟수 통계

## 산출물
- 이벤트 표시 속도 그래프
- 급가속 / 급제동 이벤트 개수
